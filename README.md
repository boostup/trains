# TrainBoard (refactor)

This repo is a refactor scaffold of the original TrainBoard prototype. It splits styles and logic into modules and adds a basic dev workflow.

Quick start (after installing dependencies):

```bash
npm install
npm run dev
```

Set your SNCF API key as an environment variable for Vite:

```bash
export VITE_SNCF_API_KEY="your-key-here"
```

Run tests:

```bash
npm test
```
